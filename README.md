# NODEPOP

## WEB-API/Node.js / MongoDB

### Install dependencies

```
npm install
```

### Seed Data Base

```
npm run installDB
```

### Run dev

```
npm run dev
```

### Run the api

```
npm run start
```

## API REQUEST

### Show all advertisements

```
http://localhost:3000/api/advertisements
```
